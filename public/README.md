# Public Assets

## Church Background Image

Please save the church image as `church-bg.jpg` in this directory.

The image should be:
- Filename: `church-bg.jpg`
- Location: `/public/church-bg.jpg`
- Recommended size: 1920x1080 or similar high-resolution format
- Format: JPG or PNG

Once added, the image will automatically appear as the background on the homepage.
