# Bible Exhibition Registration - User Guide

## How to Use the System

### For First-Time Registration

1. **Go to the Website**
   - Visit the home page
   - Click "Register Your Group" button

2. **Fill Registration Form**
   - Enter your name
   - Enter church name
   - Select preferred date
   - Enter total people and language split
   - **Enter phone number** (REQUIRED - you'll need this later!)
   - Optionally add email

3. **Submit & Get Confirmation**
   - You'll receive a registration number
   - You'll see a QR code
   - You'll see your assigned time slots
   - **Save your phone number** - you'll need it to view/manage registration

### How to View Your Registration Later

**Option 1: From Home Page**
1. Go to home page
2. Click "View My Registration" button
3. Enter your phone number
4. See all your registrations

**Option 2: Direct Link**
1. Go to `/lookup` page directly
2. Enter your phone number
3. See all your registrations

**Option 3: From Confirmation Page**
1. After registering, click "View My Registrations"
2. Enter your phone number
3. See all your registrations

### What You Can Do on Lookup Page

Once you enter your phone number, you can:
- ✅ View all your registrations
- ✅ See registration details (date, time, people count)
- ✅ Access your QR code anytime
- ✅ Cancel registration (if not checked in yet)
- ✅ See check-in status

### QR Code Usage

**The QR code is for check-in at the exhibition, NOT for lookup!**

- **At Exhibition:** Show QR code for quick check-in
- **Before Exhibition:** Use phone number to lookup registration
- **Lost QR Code?** No problem! Use lookup page to access it again

## User Flows Explained

### Flow 1: Register → Save Phone → Done
```
1. Register with phone number
2. Get QR code (save screenshot or remember phone)
3. On exhibition day: Show QR code at entrance
```

### Flow 2: Register → Forgot Details → Lookup
```
1. Register with phone number
2. Later, forgot registration details
3. Go to home page → Click "View My Registration"
4. Enter phone number
5. See all details + QR code
```

### Flow 3: Register → Need to Cancel
```
1. Register with phone number
2. Plans changed, need to cancel
3. Go to lookup page
4. Enter phone number
5. Click "Cancel" button on registration
6. Slots freed up for others
```

### Flow 4: Register Multiple Times
```
1. Register for June 6 with phone: +91 9876543210
2. Later, want to register for June 13 too
3. Register again with same phone: +91 9876543210
4. Both registrations linked to your phone
5. Lookup shows both registrations
```

## Important Notes

### Phone Number
- **Required for all registrations**
- Use the same format each time (e.g., +91 9876543210)
- You can register up to 2 times with same phone (different dates)
- Cannot register twice for the same date with same phone

### QR Code
- **For check-in at exhibition** (scan at entrance)
- **NOT for looking up registration** (use phone number instead)
- Can be accessed anytime via lookup page
- Remains valid until check-in

### Cancellation
- ✅ Can cancel before check-in
- ❌ Cannot cancel after check-in
- Cancellation frees up slots immediately
- No penalty for cancellation

### Multiple Registrations
- Same phone can have max 2 registrations
- Must be for different dates
- All registrations shown in lookup page
- Each has its own QR code

## Common Questions

**Q: I lost my QR code, what do I do?**
A: Go to lookup page, enter your phone number, and access your QR code again.

**Q: How do I change my registration?**
A: Cancel the old registration (if not checked in) and create a new one.

**Q: Can I register for multiple dates?**
A: Yes! You can register up to 2 times with the same phone number for different dates.

**Q: I forgot my registration number, how do I find it?**
A: Go to lookup page and enter your phone number. You'll see all your registrations.

**Q: Can someone else use my phone number to cancel my registration?**
A: Yes, so keep your phone number private. Only share with trusted people.

**Q: What if I used a different phone number format?**
A: Use the exact same format you used during registration. For example, if you registered with "+91 9876543210", use that exact format in lookup.

**Q: Can I edit my registration details?**
A: Currently, you need to cancel and re-register. Contact admin if you need help.

## Tips for Best Experience

1. **Save Your Phone Number**
   - Use the same phone number for all registrations
   - Remember the format you used

2. **Take a Screenshot**
   - Screenshot the confirmation page
   - Save the QR code image
   - Keep registration number handy

3. **Check Before Exhibition**
   - Use lookup page to verify details
   - Confirm your time slots
   - Make sure you have QR code ready

4. **Arrive Early**
   - Come 10 minutes before your slot
   - Have QR code ready on phone
   - Follow batch sequence if multiple slots

## Navigation Guide

### From Home Page
- **"Register Your Group"** → Registration form
- **"View My Registration"** → Lookup page (enter phone)

### From Registration Form
- **Blue info banner** → Links to lookup page
- **"← Back"** → Returns to home page

### From Confirmation Page
- **"Back to Home"** → Home page
- **"View My Registrations"** → Lookup page
- **"Save / Share"** → Save or share confirmation

### From Lookup Page
- **"← Back to Home"** → Home page
- **"View QR Code"** → Opens QR code page
- **"Cancel"** → Cancels registration (if not checked in)
- **"Register Now"** → Registration form (if no registrations found)
